# Analytics & Metrics

## Dashboard Design
- Select 5-7 key metrics per business function
- Real-time vs daily/weekly cadence
- Traffic light thresholds for each metric

## Funnel Analysis
1. Map full funnel: Awareness → Interest → Consideration → Conversion → Retention
2. Measure drop-off at each stage
3. Identify bottlenecks with highest drop-off
4. A/B test improvements at bottleneck stages

## Attribution Models
- Last-touch (simple but misleading)
- Multi-touch (weighted across journey)
- Channel-specific ROI calculation

## Key Metrics by Function
| Function | Metrics |
|----------|---------|
| Growth | DAU/MAU, sign-ups, activation rate |
| Revenue | MRR, ARR, ARPU, expansion revenue |
| Retention | Churn rate, NPS, DAU/MAU ratio |
| Efficiency | CAC, LTV:CAC ratio, payback period |
