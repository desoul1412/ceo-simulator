# Pre-Flight Documentation Check

**Directive:** Read `brain/00-Index.md` before coding. If a feature lacks a spec document in `/wiki/`, write the spec first before implementing.

**Why:** Ensures all work is documented and traceable. Prevents building without a plan.

**Scope:** Global
