# No Hallucination

**Directive:** Use Tavily MCP for market data, competitor info, and real-world facts. Use Context7 MCP for API/framework syntax verification. Never fabricate data, URLs, or code examples from training data.

**Why:** Production systems must be built on verified facts.

**Scope:** Global
