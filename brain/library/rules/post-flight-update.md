# Post-Flight Documentation Update

**Directive:** Upon task completion, update the relevant spec file in `brain/wiki/` and append actions to `brain/changelog.md`. Do not ask permission — just do it.

**Why:** Keeps the knowledge base current. Every change is traceable.

**Scope:** Global
