# Customer Success

## User Onboarding
1. **First-run experience** — Guide to first value moment (under 5 minutes)
2. **Activation metrics** — Define what "activated" means (e.g., created first project)
3. **Progressive onboarding** — Reveal features as user is ready, not all at once
4. **Health scoring** — Track engagement signals per user/account

## Retention
- Cohort analysis: Day-1, Day-7, Day-30 retention rates
- Churn indicators: declining usage, support tickets, feature requests ignored
- Intervention triggers: automated emails when engagement drops
- Win-back sequences for churned users

## Feedback Loops
- In-app feedback widgets at key moments
- NPS surveys (quarterly)
- User interviews (monthly, 5 users minimum)
- Feature request tracking with voting/prioritization
- Close the loop: notify users when their feedback ships

## Support Knowledge Base
- Categorize by user journey stage
- FAQ format with clear, direct answers
- Include troubleshooting decision trees
- Track most-viewed articles to identify pain points
