# UI/UX Pro Max

AI-powered design intelligence with comprehensive UI/UX guidance.

## Capabilities
- **67 UI Styles** — Glassmorphism, Claymorphism, Minimalism, Brutalism, Neumorphism, Bento Grid, Dark Mode, AI-Native UI, etc.
- **161 Industry-Specific Rules** — Product category reasoning (SaaS, healthcare, e-commerce, gaming, etc.)
- **161 Color Palettes** — Industry-aligned with product types
- **57 Font Pairings** — Curated Google Fonts combinations
- **25 Chart Types** — Dashboard/analytics recommendations
- **99 UX Guidelines** — Best practices and anti-patterns
- **Design System Generator** — Analyzes requirements and generates complete tailored design systems

## When to Use
- Starting a new UI component or page
- Choosing color schemes and typography
- Designing dashboards with charts
- Creating consistent design systems
- Evaluating UI patterns for specific industries

## Installation
```bash
npm install -g uipro-cli
uipro init --ai claude
```

## Usage
Describe your design needs naturally. The skill auto-activates for UI/UX work and provides:
- Style recommendations based on project type
- Color palette suggestions
- Font pairing recommendations
- Component layout patterns
- Accessibility checks
