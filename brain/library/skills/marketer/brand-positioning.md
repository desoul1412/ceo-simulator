# Brand Positioning

## Positioning Statement
```
For [target audience] who [need/pain point],
[Product] is a [category] that [key benefit].
Unlike [competitor], we [differentiator].
```

## Messaging Framework
1. **Tagline** — One-line value proposition
2. **Elevator pitch** — 30-second explanation
3. **Key messages** — 3-5 supporting points with proof
4. **Objection responses** — Common pushbacks with counters

## Brand Voice
- Define personality (e.g., expert but approachable, bold but precise)
- Vocabulary: words to use vs avoid
- Tone adaptation per channel (formal on LinkedIn, casual on Twitter)

## Competitive Differentiation
- Feature comparison matrix
- Unique value propositions per segment
- "Why us" narratives for sales and marketing
