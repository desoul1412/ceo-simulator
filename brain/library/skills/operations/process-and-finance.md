# Process & Finance Operations

## Standard Operating Procedures (SOPs)
- Document every repeatable process
- Format: trigger → steps → expected output → exception handling
- Store in `brain/wiki/` for Obsidian vault access
- Review quarterly for accuracy

## Budget Management
- Token budget allocation per agent per month
- Cost forecasting based on historical usage
- Burn rate tracking and alerts at 80% threshold
- ROI calculation per feature: cost to build vs value delivered

## Financial Reporting
- Weekly: token spend by agent, tickets processed
- Monthly: total cost, cost per feature, budget vs actual
- Quarterly: trend analysis, budget reallocation recommendations

## Automation Strategy
1. Identify manual, repeatable processes
2. Estimate time saved by automating
3. Prioritize by ROI (time saved / effort to automate)
4. Build and deploy automation
5. Measure actual time saved vs estimate

## Team Capacity Planning
- Agent utilization rate (working hours / available hours)
- Workload balancing across agents
- Bottleneck identification (which agent is always blocked?)
- Recommendations: hire more agents or redistribute work
