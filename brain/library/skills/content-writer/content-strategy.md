# Content Strategy

## Topic Clusters
1. Identify 3-5 pillar topics aligned with product value
2. Map 5-10 subtopics per pillar
3. Create pillar page (comprehensive, 2000+ words)
4. Create cluster articles linking back to pillar
5. Internal linking between related clusters

## Editorial Calendar
- Weekly cadence minimum (blog/social/email)
- Theme months aligned with product roadmap
- Content types: blog, social, email, video script, docs update
- Assignment and deadline tracking

## Content Repurposing
One piece of content becomes:
- Blog post → Twitter/X thread
- Blog post → LinkedIn article (reformatted)
- Blog post → Email newsletter section
- Blog post → Social media carousel
- Key stats → Infographic
- Tutorial → Video script

## Email Sequences
- Welcome (day 0): value proposition + quick win
- Onboarding (days 1-7): feature education
- Engagement (ongoing): tips, use cases, updates
- Re-engagement (30+ days inactive): win-back offer
